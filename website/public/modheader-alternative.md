# Header Override: ModHeader Alternative

Header Override is an open-source, privacy-first ModHeader alternative for modifying HTTP request headers, response headers, request cookies, and response cookies with local browser rules.

## Overview

Header Override is built for developers, QA engineers, support engineers, and technical users comparing ModHeader alternatives for API debugging, staging environments, feature flags, cookies, and request-specific testing.

Rules are stored locally in the user's browser and applied only to matching requests. Header Override does not use analytics, tracking, advertising, remote code, external servers, or third-party data sharing.

## Why Use Header Override As A ModHeader Alternative

- Modify HTTP request headers with scoped URL rules.
- Modify HTTP response headers from the same browser extension popup.
- Override request Cookie headers.
- Override response Set-Cookie headers and common cookie attributes.
- Keep up to five local profiles for projects, clients, environments, and test accounts.
- Import and export selected profiles as local JSON files.
- Use Chrome, Edge, and Firefox.
- Store rules locally in browser extension storage.
- Review the public source code on GitHub.
- Use a focused browser extension with no analytics, no advertising, no tracking, and no remote code.

## FAQ

### Is Header Override an alternative to ModHeader?

Yes. Header Override is a browser extension for users who need to modify HTTP headers and cookies with scoped local rules. It focuses on a small, open-source, privacy-first workflow rather than a broad debugging platform.

### Can Header Override modify request and response headers?

Yes. Header Override supports request header rules and response header rules, and each rule can be scoped to browser-supported URL filter patterns.

### Does Header Override store rules on a server?

No. Header Override stores rules locally in browser extension storage and does not transmit your rules, browsing activity, or website content to the developer or third parties.
